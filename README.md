# Catch the Falling Object

## Project Overview:
Catch the Falling Object is a simple 2D Java-based game developed using Java Swing. In this game, the player controls a catcher (basket) that moves horizontally at the bottom of the screen. The main goal is to catch falling objects that drop randomly from the top of the screen. Each successful catch increases the player's score. The game becomes more challenging as the speed of falling objects increases.

This project is designed to apply basic game development concepts using Java along with Object-Oriented Programming (OOP) principles and GUI development.

## Objectives:
- To develop a simple interactive game using Java.
- To understand and apply Object-Oriented Programming (OOP) concepts.
- To learn Java Swing for GUI-based application development.
- To implement basic game logic such as movement, collision detection, and scoring system.
- To improve logical thinking and problem-solving skills.

## Features:
- Player-controlled catcher (left/right movement)
- Random falling objects from the top
- Score tracking system
- Collision detection between catcher and objects
- Game over condition when required
- Simple and user-friendly interface

## Working Procedure:
1. The game window opens with a catcher placed at the bottom.
2. The player moves the catcher using keyboard arrow keys (left/right).
3. Objects are generated randomly at the top of the screen.
4. These objects fall vertically with constant or increasing speed.
5. If the catcher catches an object, the score increases.
6. If an object is missed (depending on game rules), the game may end or reduce lives.
7. The game continues until a game-over condition is reached.
