# Week 1: Basic Game Window

##  Objective
The main objective of Week 1 is to create a basic game window using Java Swing. This includes setting up a `JFrame` window, adding a `JPanel` as the game screen, and displaying an empty but properly structured window.

---

##  Tasks Completed

### 1. Create JFrame (Main Window)
- A `JFrame` is created as the main application window.
- Window title is set for identification.
- Default close operation is configured to close the program when the window is closed.

### 2. Create Game Screen (JPanel)
- A custom `JPanel` is created to act as the game screen.
- This panel is added inside the JFrame.
- It will later be used for drawing game graphics.

### 3. Set Window Size
- The window size is defined (width and height).
- The window is set to be non-resizable for consistent gameplay layout.

### 4. Set Background Color
- A background color is applied to the game panel.
- This helps visually confirm that the panel is working correctly.

### 5. Run Basic Empty Game Screen
- The program is executed to display an empty game window.
- No game logic or objects are added yet.
- Only the basic structure is tested successfully.

---

##  Output
- A simple game window opens on the screen.
- The window contains a blank game panel with a visible background color.
- No animations or controls are implemented in this week.

---

##  Technologies Used
- Java
- Swing (JFrame, JPanel)

---

##  Result
A functional base window for a game project has been successfully created. This will be used as the foundation for future game development steps.
