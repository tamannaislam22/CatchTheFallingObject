# Week 2: Player Setup

## Objective

The objective of Week 2 is to create a player (catcher) and implement keyboard controls for left and right movement while ensuring the player remains inside the game window.

## Tasks Completed

- Created a player (catcher) using a rectangle.
- Positioned the player at the bottom of the game screen.
- Implemented keyboard controls using the Left Arrow and Right Arrow keys.
- Restricted the player's movement within the game window boundaries.
- Added a welcome message ("Catch the Falling Objects Game Started!") that appears for 3 seconds before the game begins.

## Technologies Used

- Java
- Java Swing
- AWT Graphics
- KeyListener
- Timer

## Output

- The game window opens successfully.
- A welcome message is displayed for 3 seconds.
- After the message disappears, the player (catcher) appears at the bottom of the screen.
- The player moves left and right using the keyboard arrow keys.
- The player cannot move outside the game window.
